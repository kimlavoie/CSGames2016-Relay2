# Problème #2

Vous devez créer un interpréteur de Brainf\*ck. Votre programme doit prendre en argument le chemin vers un fichier contenant un programme Brainf\*ck valide et l'interpréter correctement.

Vous devez également fournir un fichier contenant un programme affichant "J'ai réussi!" en Brainf\*ck pour pouvoir tester votre interpréteur.

Vous pouvez utiliser les langages suivants: C/C++/Python/Java.
